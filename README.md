# 📝 Todo Manager

A modern, feature-rich task management application with categories and inline editing.

## 🌐 Live Demo

[View Todo Manager](https://RavenGames.github.io/todo-manager/)

## 🚀 Features

- **Task Management**: Add, edit, delete, and mark tasks as complete
- **Categories**: Organize tasks by Work, Personal, Learning, or custom categories
- **Inline Editing**: Edit tasks directly without modal dialogs
- **Local Storage**: Persistent data storage that survives browser sessions
- **Dark/Light Theme**: Toggle between themes for different preferences
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile
- **Smooth Animations**: Polished UI with smooth transitions
- **Task Filtering**: Filter tasks by completion status and category

## 🛠️ Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Storage**: Browser localStorage API
- **Styling**: Modern CSS with CSS Grid and Flexbox
- **Icons**: Font Awesome icon library
- **No External Dependencies**: Pure vanilla JavaScript

## 🚀 Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Start adding tasks using the input field
4. Use categories to organize your tasks
5. Toggle theme using the moon/sun icon

## 🌟 Key Features Explained

### Task Operations

- **Add Tasks**: Type in the input field and click Add or press Enter
- **Edit Tasks**: Click directly on task text to edit inline
- **Complete Tasks**: Click the checkbox to mark as done
- **Delete Tasks**: Click the trash icon to remove tasks

### Organization

- **Categories**: Assign tasks to Work, Personal, or Learning
- **Filtering**: Filter by All, Active, or Completed tasks
- **Persistence**: All data is automatically saved to localStorage

### User Experience

- **Responsive Design**: Adapts to any screen size
- **Theme Toggle**: Switch between light and dark modes
- **Smooth Animations**: Polished interactions and transitions
- **Keyboard Support**: Full keyboard navigation support
